# World Chain Coins Changelog

This changelog documents important public developments across the World Chain Coins ecosystem.

It is intended to provide a clear overview of major launches, platform updates, new nations, community milestones and ecosystem expansions.

## 2026

### July 2026

- Public World Chain Coins GitHub documentation launched
- Official ecosystem links and verified contract addresses published
- Public roadmap added
- Security information added
- Nations documentation added
- Multilingual platform expanded
- Translation system improved
- Additional ecosystem documentation prepared

### June 2026

- China unlocked by the community
- World Chain Coins TikTok launched
- Telegram bot introduced
- New nation pages and ecosystem content added
- Community voting expanded
- Collection system continued development
- Public project updates expanded

### May 2026

- Turkey became the first nation unlocked through community voting
- Community features expanded
- Additional live nation coins added
- Nation identities and Country Kids universe continued growing
